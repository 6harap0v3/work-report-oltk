# 🗂 Рабочий журнал — Telegram бот

Бот для учёта рабочих выездов. Просто напишите что сделали — бот сохранит запись.

## Команды

| Команда | Описание |
|---------|----------|
| `/start` | Начало работы |
| `/list` | Последние 10 записей |
| `/excel` | Скачать все записи в CSV |
| `/delete` | Удалить последнюю запись |
| `/help` | Помощь |

## Пример записи

```
27 апреля работал на Звёздном 44, подключил абонента, с 11 до 18, взял 10000, из них мои 3000
```

## Деплой на Railway

1. Форкните / загрузите репозиторий на GitHub
2. Зайдите на [railway.app](https://railway.app)
3. New Project → Deploy from GitHub repo
4. Добавьте переменную окружения: `BOT_TOKEN = ваш_токен`
5. Deploy — готово!
